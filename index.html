GameHub

GameHub is an online store focusing on selling video games. It was created using basic html and CSS. Allowing the user to select a single product and simulate adding it to your cart and buying it. Desgined to appear to a young adult video game playing audience. It is a demonstration in design, responsiveness and skills in html and css.
